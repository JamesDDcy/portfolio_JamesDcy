SUPERPOT

The SuperPot is an Internet of Things project, which aims to create a self-sufficient system that automatically waters crops, adjusts shading based on weather conditions, and allows remote monitoring and control via the Internet.

Main features includes a moisture sensor-dependent automatic watering system to maintain ideal soil moisture levels ,a light sensor-dependent shade mechanism to protect plants from harsh sunlight ,and a remote control watering and shade mechanism as well as monitoring via the Internet for convenience and accessibility.

The project won a 'Best Exhibit Award' as it was presented in the IOT Cup UPD DCS 2024.
